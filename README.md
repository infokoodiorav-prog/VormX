# VORMX — Boxing/Fitness Landing Page

Modern responsive landing page for a fictional fitness gym brand.  
Designed to simulate a real gym membership marketing website with a strong focus on conversions, pricing clarity, and structured training programs.

---

## 🚀 Overview

VormX is a gym and training concept landing page built to simulate a real-world fitness business website.

The project focuses on:
- membership sales funnels
- structured training programs
- conversion-driven UI design
- clean, modern fitness branding

---

## 🎯 Key Features

- Fully responsive landing page (mobile-first)
- Hero section with strong marketing message
- Membership pricing cards
- Weekly training schedules (boxing + gym split)
- Statistics / social proof section
- Call-to-action conversion section
- Multi-language system (ET / EN ready)
- Modular `data-i18n` translation structure

---

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Custom i18n translation system

## 🏋️ Project Concept

This project simulates a real fitness gym brand website.

The goal was to replicate modern gym marketing sites where:
- users are pushed toward membership plans
- trust is built through stats and structure
- training schedules increase credibility
- CTA sections drive conversions

Inspired by modern fitness brands and gym SaaS landing pages.

---

## 🌍 Language Support

The site includes a simple translation system using `data-i18n` attributes.

Supported languages:
- Estonian (default)
- English
